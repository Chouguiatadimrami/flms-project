📌 Project Description

A full-stack web application that manages a faculty library system.
It allows students and faculty to browse and borrow books, while librarians and admins manage resources and users.

⚙️ Tech Stack

- Frontend: React.js
- Backend: Node.js + Express
- Database: MongoDB

🔑 Features

- Authentication (login/register)
- Book catalog (search, filter)
- Borrow & return system
- Role-based access (Student / Faculty / Librarian / Admin)

🚀 Setup Instructions

Backend

cd backend
npm install
npm start

Frontend

cd frontend
npm install
npm start

📂 Project Structure

- backend/ → API & business logic
- frontend/ → UI application
- docs/ → documentation
